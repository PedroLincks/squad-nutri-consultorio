# Formulas Classicas de Copywriting

> Formulas extraidas dos livros. Usadas pela Zoe para estruturar copies.

## AIDA

- **Attention:** Capturar atencao com headline forte
- **Interest:** Gerar interesse com fatos, dados, historia
- **Desire:** Criar desejo mostrando beneficios e transformacao
- **Action:** Chamar para acao com CTA claro

## PAS

- **Problem:** Apresentar o problema
- **Agitate:** Agitar a dor, mostrar consequencias
- **Solution:** Apresentar a solucao

## BAB

- **Before:** Situacao atual (com dor)
- **After:** Situacao desejada (com prazer)
- **Bridge:** Seu produto/servico como ponte

## 4U's

- **Useful:** Util para o leitor
- **Urgent:** Senso de urgencia
- **Unique:** Unico, diferente
- **Ultra-specific:** Extremamente especifico

## Star-Story-Solution

- **Star:** Personagem principal (o prospect ou alguem como ele)
- **Story:** Historia de transformacao
- **Solution:** Seu produto como a solucao

## PPPP (Picture-Promise-Prove-Push)

- **Picture:** Pintar cenario
- **Promise:** Prometer resultado
- **Prove:** Provar com evidencias
- **Push:** Empurrar para acao
